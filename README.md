Верстка личного сайта портфолио.

### Примененные технологии в проекте:

* HTML5
* CSS3
* JavaScript, jQuery
* LESS
* Bootstrap 3
* validate.js
* fancybox

### Что внутри:

* Верстка
* Мобильная адаптивность
* Фильтр работ
* Резюме разработчика
* Работы в портфолио
